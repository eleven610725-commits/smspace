# 💕 SMSpace - 双人亲密关系空间

一个简洁优雅的移动端网页应用，为情侣打造的私密专属空间。

## ✨ 主要功能

### 🔐 用户系统
- 📧 邮箱注册/登录
- 👤 个人资料管理
- 🔑 安全的密码存储

### 💑 配对与邀请
- 生成唯一邀请码
- 邀请伴侣加入
- 配对确认机制

### 💌 亲密空间
- **爱情誓言** - 记录你们的誓言
- **共享相册** - 珍藏美好回忆
- **纪念日计数** - 记录重要日期

### 📍 每日打卡
- 🌅 早上打卡
- 🌙 晚上打卡
- 😊 心情记录
- 📊 月度统计

### 🎯 甜蜜任务
- 创建任务清单
- 完成度追踪
- 双人互动

### 📔 每日日记
- ✍️ 记录每天的想法
- 😍 心情表达（5种情绪）
- 📚 历史日记查看
- 💭 隐私保护

## 🎨 主题配色

集成5套预设主题，可随时切换：
- 💗 粉色（默认）
- 💙 蓝色
- 💜 紫色
- 💚 绿色
- ⬛ 深色

## 🚀 快速开始

### 无需安装依赖！

这是一个**零依赖的单文件应用**，完全使用原生 HTML + CSS + JavaScript 编写。

### 本地运行
```bash
# 方法1：直接用浏览器打开
# 在文件管理器中找到 index.html，双击打开

# 方法2：用 Python 启动本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

### 一键部署到 Vercel（推荐）

1. **访问 Vercel**: https://vercel.com
2. **点击 "New Project"**
3. **导入这个 GitHub 仓库**
4. **点击 "Deploy"** - 自动完成！

部署完成后，你会获得一个 URL，分享给伴侣即可使用。

## 💾 数据存储

目前使用 **浏览器本地存储（LocalStorage）**：
- ✅ 完全免费
- ✅ 数据存储在本地，隐私安全
- ✅ 无需服务器
- ⚠️ 每个浏览器独立存储（同一设备不同浏览器数据不同步）

### 如需云同步（可选）

将 Firebase 配置替换为你自己的：
1. 访问 [Firebase Console](https://console.firebase.google.com)
2. 创建新项目
3. 复制你的 Firebase 配置
4. 替换代码中的这行：
```javascript
const firebaseConfig = {
    // 你的 Firebase 配置
};
```

## 📱 功能说明

### 注册登录
- 支持邮箱和密码
- 密码至少6位
- 数据本地加密存储

### 邀请伴侣
1. 点击"邀请"标签
2. 获得唯一邀请码
3. 复制分享给伴侣
4. 伴侣输入码加入

### 编辑誓言
1. 配对后点击"编辑誓言"
2. 输入你们的誓言
3. 自动保存并显示

### 每日打卡
1. 早上和晚上各打卡一次
2. 选择心情（可选）
3. 添加备注
4. 自动统计月度次数

### 记录日记
1. 选择今天的心情
2. 写下想法和感受
3. 自动保存日期
4. 伴侣可以看到历史日记

### 创建任务
1. 输入甜蜜任务
2. 完成后标记
3. 可以删除已完成任务

### 记录纪念日
1. 输入重要日期
2. 事件名称（如：第一次见面）
3. 自动计算距离天数

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **数据存储**: LocalStorage（本地） / Firebase（可选云存储）
- **部署**: Vercel / Netlify / 任何静态托管服务

## 📊 数据结构

所有数据存储在浏览器 LocalStorage 中：

```
smspace_users              // 用户账户
smspace_couples            // 配对信息
smspace_vow_*              // 誓言
smspace_checkins_*         // 打卡记录
smspace_tasks_*            // 任务列表
smspace_diaries_*          // 日记
smspace_anniversaries_*    // 纪念日
smspace_theme              // 主题设置
smspace_currentUser        // 当前登录用户
```

## 🔒 隐私与安全

- ✅ 数据完全存储在本地浏览器
- ✅ 不连接任何外部服务器（除非配置 Firebase）
- ✅ 无广告、无追踪
- ⚠️ 清除浏览器数据会丧失本地数据
- 💡 建议定期备份重要数据
