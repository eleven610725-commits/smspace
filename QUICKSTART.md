## 🚀 本地测试（可选）

在部署前，可以先本地测试：

### 方法 1：直接打开（最简单）
```bash
# Windows/Mac/Linux
# 在文件管理器中找到 index.html
```

```bash

# 然后在浏览器打开：
# http://localhost:8000
```

### 方法 3：用其他服务器
```bash
# Node.js (需要安装)
npx http-server

# 或用 PHP
php -S localhost:8000
```

## 🌐 部署到 Vercel（推荐）

### 快速部署
1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 **"New Project"**
4. 导入 `smspace` 仓库
5. 点击 **"Deploy"**
6. 等待 1-2 分钟 ✅

### 获得你的 URL
部署完成后，你会看到：
```
✅ Congratulations! Your site is live.
https://smspace-xxx.vercel.app
```

**保存这个 URL！** 这就是你的应用链接。

## 👥 邀请伴侣使用

### 第一次设置

#### 你（User A）
1. 打开应用 URL
2. 点击 **"注册"**
3. 输入：
   - 昵称：你的名字
   - 邮箱：你的邮箱
   - 密码：设置密码（6位以上）
4. 点击 **"注册"**
5. 登录成功 ✅

#### 伴侣（User B）
1. 打开同一个应用 URL
2. 点击 **"注册"**
3. 输入：
   - 昵称：伴侣的名字
   - 邮箱：伴侣的邮箱
   - 密码：设置密码
4. 点击 **"注册"**
5. 登录成功 ✅

### 配对

#### 你（User A）发送邀请码
1. 点击底部导航的 **"💌 邀请"** 标签
2. 看到 **"你的邀请码"** 框
3. 点击 **"📋 复制邀请码"**
4. 分享给伴侣（微信、QQ、短信等）

#### 伴侣（User B）输入邀请码
1. 点击底部导航的 **"💌 邀请"** 标签
2. 看到 **"加入伴侣的亲密空间"** 区域
3. 粘贴你发送的邀请码
4. 点击 **"加入"**
5. 配对成功 ✅

## 💕 开始使用各功能

### 🏠 首页

**功能：**
- 查看配对信息和相伴天数
- 编辑爱情誓言
- 进行每日打卡
- 查看打卡统计

**怎么用：**
1. 登录后自动进入首页
2. 看到对方的名字和相伴天数
3. 点击 **"编辑誓言"** 添加你们的誓言
4. 早上和晚上各点击打卡按钮
5. 选择心情，添加备注（可选）

### 🎯 甜蜜任务

**功能：**
- 创建任务
- 上传图片/视频完成证明
- 反馈和确认

**流程：**
```
创建者 → 创建任务
    ↓
被分配者 → 查看任务
    ↓
被分配者 → 上传完成证明（图片/视频）
    ↓
创建者 → 确认完成
    ↓
任务标记为已完成 ✅
```

**具体操作：**
1. 点击 **"🎯 任务"** 标签
2. 输入任务标题和描述
3. 点击 **"添加任务"**
4. 伴侣可以看到任务，点击任务查看详情
5. 伴侣点击 **"📝 添加反馈"**
6. 上传图片或视频作为完成证明
7. 写下说明
8. 点击 **"提交反馈"**
9. 你（创建者）看到反馈后，点击 **"确认完成 ✓"**

### 📔 每日日记

**功能：**
- 记录每天的想法
- 记录心情（5种表情）
- 上传图片
- 伴侣可以看到你的日记

**具体操作：**
1. 点击 **"📔 日记"** 标签
2. 选择今天的心情（😊 😍 😌 😢 😠）
3. 在文本框写下想法
4. 可选：点击 **"上传图片"** 添加照片
5. 点击 **"保存日记"**
6. 日记会显示在 **"历史日记"** 区域
7. 伴侣也能看到你的日记

### 💑 亲密空间

**功能：**
- 查看配对信息
- 添加纪念日
- 自动计算距离天数

**具体操作：**
1. 点击 **"💑 空间"** 标签
2. 看到配对信息
3. 输入重要日期（如：第一次见面）
4. 输入事件名称
5. 点击 **"添加纪念日"**
6. 日期会自动显示剩余天数

### ⚙️ 设置

**功能：**

3. 点击喜欢的颜色

## 🎓 使用小贴士

### 📸 媒体上传建议

**图片：**
- 推荐格式：JPG, PNG
- 最大大小：10MB
- 建议分辨率：1920×1440 或更小

**视频：**
- 推荐格式：MP4, WebM
- 最大大小：50MB
- 建议时长：<5 分钟
- 建议分辨率：720p 或更小

**压缩工具：**
- 图片：https://tinypng.com
- 视频：https://www.online-convert.com

### 🌐 网络条件

- 上传媒体时需要稳定网络
- 建议使用 WiFi 上传较大文件
- 首次加载可能需要 3-5 秒

### 💡 最佳实践

1. **定期打卡** - 养成每天打���的习惯
2. **坚持写日记** - 记录生活中的美好时刻
3. **创建有意义的任务** - 比如"一起看场电影"
4. **给任务拍照** - 上传完成证明更有趣
5. **定期更新誓言** - 随着时间更新你们的承诺

## 🆘 遇到问题？

### 问题排查流程

1. **刷新页面** F5 或 Ctrl+R
2. **清除浏览器缓存** (Ctrl+Shift+Delete)
3. **打开开发者工具** (F12)
4. **查看 Console 标签页**
5. **记下错误信息**
6. **在 GitHub 提交 Issue**

### 常见问题

| 问题 | 解决方案 |
|------|---------|
| 无法注册 | 检查邮箱格式，密码是否 6 位以上 |
| 无法登录 | 检查邮箱和密码是否正确 |
| 邀请码无效 | 确保邀请码没有抄错，区分大小写 |
| 无法上传图片 | 检查文件大小 <10MB，网络是否稳定 |
| 页面加载慢 | 这是正常的，首次加载需要时间 |
| 看不到伴侣的数据 | 确认已配对，刷新页面重试 |
